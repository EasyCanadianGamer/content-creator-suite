# Twitter/X Manager

Write high-performing tweets, threads, and build an engaged following on X.

---

## Authentication

Twitter/X management uses the Twitter API v2.
- Check if connected via Settings → Connectors → Twitter/X
- If not: "Connect Twitter/X in Settings → Connectors to post directly. Until then, I can write everything for you to copy and post."

---

## The X Algorithm — Key Facts

1. **Replies and quote tweets > likes** — conversation drives distribution
2. **Early engagement matters** — engagement in first 30–60 mins determines reach
3. **Twitter Blue / X Premium posts get boosted** — verified accounts get more distribution
4. **Images and videos outperform text-only** for impressions
5. **Threads get more reach than single tweets** when they drive reply activity
6. **Don't post external links in tweets** — it reduces reach (put them in replies)
7. **Consistency signals reliability** — tweeting daily trains the algorithm to show your content

## Tweet Writing

### Single Tweet Formulas
- **Hot take**: "[Contrarian opinion]. Here's why:"
- **Observation**: "I've noticed that [X]. Anyone else?"
- **Lesson**: "The biggest mistake [audience] makes is [X]."
- **Stat or insight**: "[Surprising fact or number]. What this means for [audience]:"
- **Question**: "Quick poll / genuine question to community"
- **Mini story**: Short punchy story with a lesson in under 280 chars

### Tweet Anatomy
- **Under 280 characters** for single tweets
- **Hook first** — the first line decides if they expand or scroll
- **White space** — use line breaks for readability in longer tweets
- **No hashtags** (unless trending topic join) — hashtags on X now hurt reach
- **No link in tweet** — post the link as a reply to your own tweet

## Thread Writing

Threads are the highest-reach format on X. Use this structure:

### Thread Structure
```
Tweet 1 — THE HOOK
[Bold claim, curiosity gap, or story opener]
[Thread label: "A thread 🧵" or just leave it — don't be formulaic]

Tweet 2-N — THE BODY
[Each tweet = one complete idea]
[Build on the previous]
[Use numbers: "2/ 3/ 4/" or just narrative flow]

Last tweet — THE CLOSE
[Key takeaway or synthesis]
[CTA: Follow for more, retweet, reply with your experience]
[Optional: Link to longer content as reply, not in this tweet]
```

### Thread Topics That Perform Well
- "X things I wish I knew about [topic]"
- "How I [achieved result] — a breakdown"
- "The [framework/system] that changed how I [do thing]"
- "Why [common belief] is wrong — and what to do instead"
- "Everything I know about [X] in one thread"

### Thread Best Practices
- Each tweet must standalone — someone might see just one
- Vary length: some tweets 1 line, some 3–4 lines
- Use line breaks and bullets in longer tweets
- Don't make every tweet end with a cliffhanger — it feels cheap
- End with a reply asking a question to drive engagement

## Voice & Tone Guide

X rewards distinctive voices. Help the user develop theirs:

| Voice Type | Characteristics | Best for |
|-----------|----------------|----------|
| **Expert** | Authoritative, insight-dense, confident | Thought leaders, professionals |
| **Builder/Founder** | Behind-the-scenes, honest, journey-focused | Entrepreneurs, creators |
| **Entertainer** | Witty, observational, culturally aware | Influencers, humor accounts |
| **Educator** | Clear, structured, value-first | Teachers, coaches |
| **Provocateur** | Contrarian, debate-starting, bold | Opinion leaders |

Ask: "What do you want to be known for on X?" — and build content around that identity.

## Analytics Interpretation

Key X metrics:
- **Impressions** — Total views
- **Engagements** — Any click, like, reply, retweet
- **Engagement rate** — Engagements / Impressions (1–3% = good)
- **Link clicks** — Traffic driver metric
- **Profile visits** — Interest signal
- **New followers from tweet** — Best indicator of resonance

Best-performing tweet patterns to look for:
- Which topics get the most replies? → Post more of those
- Which formats drive follows? → Double down
- What time gets most engagement? → Optimize posting schedule

## Common Workflows

### "Write me a tweet about X"
1. Get the topic and intent
2. Write 3 tweet options: direct, punchy, question-form
3. Recommend which to post based on their goals

### "Write a thread about X"
1. Get the topic, angle, and audience
2. Draft full thread (6–12 tweets)
3. Review hook — if it doesn't hook, the thread dies
4. Offer to refine any section

### "Grow my Twitter/X following"
1. Ask: current followers, niche, posting frequency
2. Analyze what's working from recent tweets (if shared)
3. Give 30-day growth plan: posting cadence, content mix, engagement habits
4. Identify 5–10 accounts to engage with in their niche

### "Reply to this tweet for me"
1. Read the tweet context
2. Write a reply that adds value, starts conversation, or shows expertise
3. Keep it punchy — best replies are under 3 lines
